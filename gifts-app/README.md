## Scroll
Flujo completo

El usuario realiza scroll.
Se obtiene la posición mediante scrollTop.
La posición se guarda en ScrollStateService.
El usuario navega a otra página.
El componente se destruye.
El servicio conserva el valor del scroll.
El usuario regresa a la página.
ngAfterViewInit() recupera el valor almacenado.
Se asigna el valor a scrollTop.
El scroll vuelve automáticamente a la posición anterior.

Este enfoque permite mantener el contexto de navegación y evita que el usuario tenga que volver a desplazarse manualmente después de cambiar de página.
